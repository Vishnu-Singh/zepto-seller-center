Project Overview

Zepto Seller Center is a comprehensive Django-based API system designed for managing seller operations. The project includes:

  • 6 Domain-Specific Apps: Products, Orders, Inventory, Shipping, Finance, Reports
  • 11 Database Models: Covering all aspects of seller operations
  • REST APIs: Full CRUD operations with pagination
  • SOAP APIs: WSDL-compliant endpoints for legacy integration
  • Admin Panels: Complete Django admin interfaces
  • Comprehensive Tests: 30+ unit tests ensuring reliability
